# chesskit

A Kotlin library for chess logic: move generation and validation, FEN/SAN/PGN/engine
LAN parsing, and a move tree supporting variations, annotations, and comments.

This is a Kotlin rewrite of [chesskit-swift](https://github.com/chesskit-app/chesskit-swift),
reimplementing its public API as closely as Kotlin allows. chesskit-swift is licensed
